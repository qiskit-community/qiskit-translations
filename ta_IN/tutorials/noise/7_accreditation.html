அங்கீகாரம் நெறிமுறை¶

அங்கீகார நெறிமுறை என்பது சத்தமில்லாத குவாண்டம் சாதனங்களின் நம்பகத்தன்மையை வகைப்படுத்த வடிவமைக்கப்பட்ட ஒரு நெறிமுறை.

ஒரு “target” ​​குவாண்டம் சுற்றுவட்டத்தை செயல்படுத்தும் சத்தமில்லாத குவாண்டம் சாதனம் கொடுக்கப்பட்டால், சாதனம் வழங்கிய வெளியீடுகளின் நிகழ்தகவு விநியோகம் மற்றும் சிறந்த நிகழ்தகவு விநியோகம் ஆகியவற்றுக்கு இடையேயான மாறுபாடு தூரத்தை AP உறுதிப்படுத்துகிறது. இந்த முறை அடிப்படையாக கொண்டதுFerracin et al, “Accrediting outputs of noisy intermediate-scale quantum devices”, https://arxiv.org/abs/1811.09709.

இந்த நூல் ignis.characterization.accireditation module ஐ எவ்வாறு பயன்படுத்துவது என்பதற்கு ஒரு எடுத்துக்காட்டு அளிக்கிறது.ஆழம் 5 இன் 4-குவிட் குவாண்டம் சுற்றுகளின் வெளியீடுகளை எவ்வாறு அங்கீகரிப்பது என்பதை இந்த குறிப்பிட்ட எடுத்துக்காட்டு காட்டுகிறது. அனைத்து சுற்றுகளும் சத்தமில்லாத ஏர் சிமுலேட்டரைப் பயன்படுத்தி இயக்கப்படுகின்றன.

[1]:

#Import general libraries (needed for functions)
import numpy as np
from numpy import random
import qiskit

#Import Qiskit classes
from qiskit import QuantumCircuit, QuantumRegister, ClassicalRegister, Aer, execute
from qiskit.providers.aer.noise import NoiseModel
from qiskit.providers.aer.noise.errors.standard_errors import depolarizing_error

#Import the accreditation functions.
from qiskit.ignis.verification.accreditation import AccreditationFitter,AccreditationCircuits

நெறிமுறைக்கு உள்ளீடு¶

ஒரு target circuit இன் வெளியீடுகளை AP அங்கீகரிக்க முடியும் 1) உள்ளீடாக எடுத்துக்கொள்கிறது:math: n மாநிலத்தில் குவிட்ஸ்:math:|{0}> `2) pauli-:math: `Z அடிப்படையில் ஒற்றை-குவிட் அளவீடுகளுடன் முடிவடைகிறது 3) ஆனது:math:` m` “bands”, ஒவ்வொரு இசைக்குழுவிலும் ஒரு சுற்று ஒற்றை-குவிட் வாயில்கள் மற்றும் ஒரு சுற்று கட்டுப்படுத்தப்பட்ட- :math: Z வாயில்கள் உள்ளன. ** பொறி சுற்றுகள் **, ஒரு கிளாசிக்கல் கணினியில் திறமையாக உருவகப்படுத்தக்கூடிய சுற்றுகள் மற்றும் சாதனத்தின் சரியான செயல்பாட்டைக் காண அதன் வெளியீடுகள் பயன்படுத்தப்படுகின்றன என்பதன் மூலம் அங்கீகாரம் செய்யப்படுகிறது.

இப்போது ஒரு இலக்கு குவாண்டம் சுற்று வரைவோம்! 4-குவிட்ஸ் GHZ நிலைகளை உருவாக்க மற்றும் அளவிட எளிய சுற்றுடன் தொடங்குகிறோம்.

[2]:

# Create a Quantum Register with n_qb qubits.
q_reg = QuantumRegister(4, 'q')
# Create a Classical Register with n_qb bits.
c_reg = ClassicalRegister(4, 's')
# Create a Quantum Circuit acting on the q register
target_circuit = QuantumCircuit(q_reg, c_reg)

target_circuit.h(0)
target_circuit.h(1)
target_circuit.h(2)
target_circuit.h(3)
target_circuit.cz(0,1)
target_circuit.cz(0,2)
target_circuit.h(1)
target_circuit.h(2)
target_circuit.cz(0,3)
target_circuit.cz(1,2)
target_circuit.h(1)
target_circuit.h(2)
target_circuit.h(3)

target_circuit.measure(q_reg, c_reg)

target_circuit.draw(output = 'mpl')

அங்கீகார சுற்றுகளை உருவாக்குதல்¶

செயல்பாடு \(accreditation\_circuits\) AP, இலக்கு மற்றும் பொறிகளுக்குத் தேவையான அனைத்து சுற்றுகளையும் உருவாக்குகிறது. இது தானாகவே சீரற்ற பவுலி வாயில்களை சுற்றுகளுக்குச் சேர்க்கிறது (செயல்படுத்தல் சத்தமாக இருந்தால், இந்த சீரற்ற பவுலி வாயில்கள் சத்தத்தை பவுலி பிழைகளுக்கு குறைக்கின்றன!)

இது பட்டியலையும் தருகிறது வெளியீடுகளை பிந்தைய செயலாக்க தேவையான சரங்களின் \(postp\_list\), அத்துடன் எண் \(v\_zero\) இலக்கை செயல்படுத்தும் சுற்றுவட்டத்தைக் குறிக்கிறது.

தோராயமாக தேர்ந்தெடுக்கப்பட்ட பவுலி வாயில்கள் கொண்ட இலக்கு சுற்று இது:

[3]:

accsys = AccreditationCircuits(target_circuit)

v = 10

circ_list, postp_list, v_zero = accsys.generate_circuits(v)
circ_list[(v_zero)%(v+1)].draw(output = 'mpl')

[3]:

ஒரு தடங்கள் எப்படி இருக்கும்:

[4]:

circ_list[(v_zero+1)%(v+1)].draw(output = 'mpl')

[4]:

cz வாயில்களுக்குப் பதிலாக cx ஐப் பயன்படுத்த விருப்பமான டூக்யூபிட்கேட் வாதத்தை ஒருவர் பயன்படுத்தலாம் மற்றும் விரும்பிய சாதன இடவியலுடன் தொகுக்க, இணைப்பு வரைபடத்தை தன்னிச்சையாக மாற்றலாம் (இது இந்த விஷயத்தில் எதிர்பார்த்ததை விட அதிகமான அடுக்குகளுக்கு வழிவகுக்கும்).

சிறந்த சுற்றுகளை உருவகப்படுத்துங்கள்¶

AP ஐ செயல்படுத்தலாம்.

நாங்கள் பயன்படுத்துகிறோம் இலக்கு மற்றும் பொறி சுற்றுகளை உருவாக்க \(accreditation\_circuits\). பின்னர், இந்த சுற்றுகள் அனைத்தையும் செயல்படுத்த \(single\_protocol\_run\) என்ற செயல்பாட்டைப் பயன்படுத்துகிறோம், அனைத்து பொறிகளும் சரியான வெளியீட்டைத் திருப்பினால் மட்டுமே இலக்கின் வெளியீட்டை வைத்திருக்கும்.

[6]:

simulator = qiskit.Aer.get_backend('qasm_simulator')

test_1 = AccreditationFitter()

# Create target and trap circuits with random Pauli gates
accsys = AccreditationCircuits(target_circuit)
circuit_list, postp_list, v_zero = accsys.generate_circuits(v)


job = execute(circuit_list,
              simulator,
              shots=1)
result = job.result()
# Post-process the outputs and see if the protocol accepts
test_1.single_protocol_run(result, postp_list, v_zero)

print("Outputs of the target: ",test_1.outputs," , AP",test_1.flag,"these outputs!")

Outputs of the target:  ['0100']  , AP accepted these outputs!

சத்தம் இல்லாத நிலையில், அனைத்து பொறிகளும் எதிர்பார்த்த வெளியீட்டைத் தருகின்றன, எனவே இலக்கின் வெளியீட்டை நாங்கள் எப்போதும் ஏற்றுக்கொள்கிறோம்.

இலக்கு சுற்றுகளின் வெளியீடுகளின் மாறுபாடு தூரத்தின் மேல் வரம்பைப் பெற, நாங்கள் AP \(d\) முறை, ஒவ்வொரு முறையும் v வெவ்வேறு பொறி சுற்றுகளுடன் செயல்படுத்த வேண்டும்.

செயல்பாடு:math: bound_variation_distance பயன்படுத்தி மாறுபாடு தூரத்தின் (வி.டி) மேல்-வரம்பைக் கணக்கிடுகிறது

\[VD\leq \frac{\varepsilon}{N_{\textrm{acc}}/d-\theta}\textrm{ ,}\]

எங்கே \(\theta\in[0,1]\) தீட்டாஒரு நேர்மறையான எண் மற்றும்

\[\varepsilon= \frac{1.7}{v+1}\]

இலக்குக்கான தவறான நிலையை ஏற்றுக்கொள்வதற்கான அதிகபட்ச நிகழ்தகவு ஆகும். செயல்பாடு \(bound\_variation\_distance\) என்பதும் கட்டுப்பட்ட நம்பிக்கையை கணக்கிடுகிறது

\[1-2\textrm{exp}\big(-2\theta d^2\big)\]
[8]:

theta = 5/100

test_2.bound_variation_distance(theta)

print("AP accepted",test_2.N_acc,"out of",test_2.num_runs,"times.")
print("With confidence",test_2.confidence,"AP certifies that VD is upper-bounded by",test_2.bound)

AP accepted 20 out of 20 times.
With confidence 1.0 AP certifies that VD is upper-bounded by 0.16267942583732053

இரைச்சல் மாதிரியை வரையறுத்தல்¶

சிமுலேட்டருக்கான சத்தம் மாதிரியை நாங்கள் வரையறுக்கிறோம். கட்டுப்படுத்தப்பட்ட -\(Z\) மற்றும் ஒற்றை-குவிட் வாயில்களில் டிப்போலரைசிங் பிழை நிகழ்தகவுகளை நாங்கள் சேர்க்கிறோம்.

[9]:

noise_model = NoiseModel()

p1q = 0.003
noise_model.add_all_qubit_quantum_error(depolarizing_error(p1q, 1), 'u1')
noise_model.add_all_qubit_quantum_error(depolarizing_error(p1q, 1), 'u2')
noise_model.add_all_qubit_quantum_error(depolarizing_error(p1q, 1), 'u3')
p2q = 0.03
noise_model.add_all_qubit_quantum_error(depolarizing_error(p2q, 2), 'cx')

basis_gates = ['u1','u2','u3','cx']

நாங்கள் சத்தமில்லாத சுற்றுகளைச் செயல்படுத்துகிறோம், அவற்றின் வெளியீடுகளுக்கு அனுப்புகிறோம் \(single\_protocol\_run\).
