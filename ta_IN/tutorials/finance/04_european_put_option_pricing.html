Pricing European Put Options¶

முன்னுரை¶

வேலைநிறுத்த விலையுடன் ஒரு ஐரோப்பிய புட் விருப்பத்தை வைத்துக்கொள்வோம் :math: K மற்றும் முதிர்ச்சியில் இடத்தின் விலை:math:`S_T`கொடுக்கப்பட்ட சீரற்ற விநியோகத்தைப் பின்பற்றுகிறது.தொடர்புடைய செலுத்துதல் செயல்பாடு பின்வருமாறு வரையறுக்கப்படுகிறது:

\[\max\{K - S_T, 0\}\]

பின்வருவனவற்றில்,எதிர்பார்த்த ஊதியத்தை மதிப்பிடுவதற்கு வீச்சு மதிப்பீட்டின் அடிப்படையில் ஒரு குவாண்டம் வழிமுறை பயன்படுத்தப்படுகிறது,அதாவது, தள்ளுபடிக்கு முன் நியாயமான விலை, விருப்பத்திற்கு:

\[\mathbb{E}\left[ \max\{K - S_T, 0\} \right]\]

அத்துடன் தொடர்புடையது:math:'Delta`,அதாவது, விலையைப் பொறுத்து விருப்ப விலையின் வழித்தோன்றல்,என வரையறுக்கப்படுகிறது:

\[\Delta = -\mathbb{P}\left[S_T \leq K\right]\]

புறநிலை செயல்பாட்டின் தோராயமாக்கல் மற்றும் விருப்ப விலை நிர்ணயம் குறித்த பொதுவான அறிமுகம் மற்றும் குவாண்டம் கணினிகளில் இடர் பகுப்பாய்வு பின்வரும் ஆவணங்களில் கொடுக்கப்பட்டுள்ளது:

  • குவாண்டம் இடர் பகுப்பாய்வு. Woerner, Egger. 2018.

  • விருப்பத்தேர்வு குவாண்டம் கணினி பயன்படுத்தப்படுகிறது. Stamatopoulos et al. 2019.

நிச்சயமற்ற மாதிரி¶

ஒரு பதிவு-சாதாரண சீரற்ற விநியோகத்தை ஒரு குவாண்டம் நிலையில் ஏற்றுவதற்கு ஒரு சுற்று தொழிற்சாலையை உருவாக்குகிறோம்.ஒரு குறிப்பிட்ட இடைவெளியில் விநியோகம் துண்டிக்கப்படுகிறது:math:[text{low}, text{high}] மற்றும் தனிப்பயனாக்கப்பட்டது:math:2^n`கட்டம் புள்ளிகள், அங்கு:math:`n பயன்படுத்தப்படும் குவிட்களின் எண்ணிக்கையைக் குறிக்கிறது.சுற்று தொழிற்சாலைக்கு ஒத்த ஒற்றுமை ஆபரேட்டர் பின்வருவனவற்றை செயல்படுத்துகிறது:

\[\big|0\rangle_{n} \mapsto \big|\psi\rangle_{n} = \sum_{i=0}^{2^n-1} \sqrt{p_i}\big|i\rangle_{n},\]

எங்கே:math:p_i துண்டிக்கப்பட்டவற்றுடன் தொடர்புடைய நிகழ்தகவுகளைக் குறிக்கவும் மற்றும் தனிப்பயனாக்கப்பட்ட விநியோகம் மற்றும் எங்கே:math:i அஃபைன் வரைபடத்தைப் பயன்படுத்தி சரியான இடைவெளியில் மேப் செய்யப்படுகிறது:

\[\{0, \ldots, 2^n-1\} \ni i \mapsto \frac{\text{high} - \text{low}}{2^n - 1} * i + \text{low} \in [\text{low}, \text{high}].\]
[2]:

# number of qubits to represent the uncertainty
num_uncertainty_qubits = 3

# parameters for considered random distribution
S = 2.0 # initial spot price
vol = 0.4 # volatility of 40%
r = 0.05 # annual interest rate of 4%
T = 40 / 365 # 40 days to maturity

# resulting parameters for log-normal distribution
mu = ((r - 0.5 * vol**2) * T + np.log(S))
sigma = vol * np.sqrt(T)
mean = np.exp(mu + sigma**2/2)
variance = (np.exp(sigma**2) - 1) * np.exp(2*mu + sigma**2)
stddev = np.sqrt(variance)

# lowest and highest value considered for the spot price; in between, an equidistant discretization is considered.
low  = np.maximum(0, mean - 3*stddev)
high = mean + 3*stddev

# construct A operator for QAE for the payoff function by
# composing the uncertainty model and the objective
uncertainty_model = LogNormalDistribution(num_uncertainty_qubits, mu=mu, sigma=sigma**2, bounds=(low, high))

[3]:

# plot probability distribution
x = uncertainty_model.values
y = uncertainty_model.probabilities
plt.bar(x, y, width=0.2)
plt.xticks(x, size=15, rotation=90)
plt.yticks(size=15)
plt.grid()
plt.xlabel('Spot Price at Maturity $S_T$ (\$)', size=15)
plt.ylabel('Probability ($\%$)', size=15)
plt.show()

செலுத்துதல் செயல்பாடு¶

செலுத்துதல் செயல்பாடு முதிர்ச்சியில் அதிகரிக்கும் ஸ்பாட் விலையுடன் நேர்கோட்டில் குறைகிறது:math:S_T வேலைநிறுத்த விலைக்கு சமமான ஸ்பாட் விலைக்கு அது பூஜ்ஜியத்தை அடையும் வரை:math:K, பெரிய ஸ்பாட் விலைகளுக்கு இது பூஜ்ஜியத்திற்கு மாறாமல் இருக்கும்.செயல்படுத்தல் ஒரு ஒப்பீட்டாளரைப் பயன்படுத்துகிறது, இது ஒரு துணை குவிட்டிலிருந்து புரட்டுகிறது:math:big|0rangle to \(\big|1\rangle\) if \(S_T \leq K\), செலுத்துதல் செயல்பாட்டின் நேரியல் பகுதியைக் கட்டுப்படுத்த இந்த துணை பயன்படுத்தப்படுகிறது.

அவர் நேரியல் பகுதி பின்வருமாறு தோராயமாக மதிப்பிடப்படுகிறது. உண்மையை நாங்கள் சுரண்டிக்கொள்கிறோம்:math:sin^2(y + pi/4) approx y + 1/2 for small \(|y|\).எனவே, கொடுக்கப்பட்ட தோராயத்தை மீட்டெடுக்கும் அளவிடுதல் காரணிக்கு:math:c_text{approx} in [0, 1] and :math:`x in [0, 1]`நாங்கள் கருதுகிறோம்

\[\sin^2( \pi/2 * c_\text{approx} * ( x - 1/2 ) + \pi/4) \approx \pi/2 * c_\text{approx} * ( x - 1/2 ) + 1/2\]

for small \(c_\text{approx}\).

ஒரு ஆபரேட்டரை நாம் எளிதாக உருவாக்க முடியும் ஆக செயல்படும்

\[\big|x\rangle \big|0\rangle \mapsto \big|x\rangle \left( \cos(a*x+b) \big|0\rangle + \sin(a*x+b) \big|1\rangle \right),\]

கட்டுப்படுத்தப்பட்ட Y- சுழற்சிகளைப் பயன்படுத்துதல்.

இறுதியில், அளவிடும் நிகழ்தகவில் நாங்கள் ஆர்வமாக உள்ளோம்:math:big|1rangle`கடைசி குவிட்டில்,இது ஒத்துள்ளது:math:sin^2(a*x+b)`.மேலே உள்ள தோராயத்துடன், இது ஆர்வத்தின் மதிப்புகளை தோராயமாக மதிப்பிட அனுமதிக்கிறது.சிறியது நாம் தேர்வு செய்கிறோம்:math:c_text{approx},சிறந்த தோராயமான.எவ்வாறாயினும், அளவிடப்பட்ட ஒரு சொத்தை நாங்கள் மதிப்பிடுகிறோம் என்பதால்:math:c_text{approx}, மதிப்பீட்டு வினாடிகளின் எண்ணிக்கை:math:m அதற்கேற்ப சரிசெய்யப்பட வேண்டும்.

தோராயமான விவரங்களுக்கு, நாங்கள் குறிப்பிடுவது: குவாண்டம் இடர் பகுப்பாய்வு. Woerner, Egger. 2018.

[4]:

# set the strike price (should be within the low and the high value of the uncertainty)
strike_price = 2.126

# set the approximation scaling for the payoff function
rescaling_factor = 0.25

# setup piecewise linear objective fcuntion
breakpoints = [low, strike_price]
slopes = [-1, 0]
offsets = [strike_price - low, 0]
f_min = 0
f_max = strike_price - low
european_put_objective = LinearAmplitudeFunction(
    num_uncertainty_qubits,
    slopes,
    offsets,
    domain=(low, high),
    image=(f_min, f_max),
    breakpoints=breakpoints,
    rescaling_factor=rescaling_factor,
)

# construct A operator for QAE for the payoff function by
# composing the uncertainty model and the objective
european_put = european_put_objective.compose(uncertainty_model, front=True)

# evaluate exact expected value (normalized to the [0, 1] interval)
exact_value = np.dot(uncertainty_model.probabilities, y)
exact_delta = -sum(uncertainty_model.probabilities[x <= strike_price])

டெல்டாவை மதிப்பிடுங்கள்¶

டெல்டா எதிர்பார்த்த ஊதியத்தை விட மதிப்பீடு செய்வது சற்று எளிது.இதேபோல் எதிர்பார்க்கப்படும் ஊதியம், வழக்குகளை அடையாளம் காண ஒரு ஒப்பீட்டாளர் சுற்று மற்றும் ஒரு துணை குவிட் ஆகியவற்றைப் பயன்படுத்துகிறோம்:math:S_T leq K. இருப்பினும், இந்த நிலை உண்மையாக இருப்பதற்கான (எதிர்மறை) நிகழ்தகவில் மட்டுமே நாங்கள் ஆர்வமாக உள்ளதால், மேலதிக தோராயமின்றி வீச்சு மதிப்பீட்டில் புறநிலை குவிட்டாக இந்த துணை குவிட்டை நாம் நேரடியாகப் பயன்படுத்தலாம்.
