Qiskit வழிமுறைகளுக்கு ஒரு அறிமுகம்¶

இது Qiskit-ல் உள்ள வழிமுறைகளுக்கான அறிமுகம் மற்றும் தொடங்குவதற்கான செயல்பாட்டின் பல்வேறு அம்சங்களைப் புரிந்துகொள்ள உதவும் உயர் மட்ட கண்ணோட்டத்தை வழங்குகிறது. பிற பயிற்சிகள் கொடுக்கப்பட்ட வழிமுறைகள் மற்றும் அவற்றைப் பயன்படுத்துவதற்கான வழிகள் போன்றவற்றில் இன்னும் ஆழமான விஷயங்களை வழங்கும்.

எப்படி வழிமுறை நூலகம் வடிவமைக்கப்பட்டுள்ளது?¶

Qiskit பல 'Algorithms <https://qiskit.org/documentation/apidoc/qiskit.aqua.algorithms.html>` __ ஐ வழங்குகிறது, மேலும் அவை அவர்கள் செய்யக்கூடிய பணிக்கு ஏற்ப வகைப்படுத்தப்படுகின்றன. உதாரணமாக, ஒரு ஆபரேட்டரின் மிகச்சிறிய ஈஜென் மதிப்பைக் கண்டுபிடிக்க Minimum Eigensolvers, எடுத்துக்காட்டாக, வேதியியல் ஹாமில்டோனியனின் நில நிலை ஆற்றல் அல்லது ஐசிங் ஹாமில்டோனியனாக வெளிப்படுத்தப்படும்போது தேர்வுமுறை சிக்கலுக்கான தீர்வு. இயந்திர கற்றல் வகைப்பாடு சிக்கல்களுக்கு Linear Solvers மற்றும் நிதி பயன்பாடுகளில் பயன்படுத்தக்கூடிய மதிப்பு மதிப்பீட்டிற்கான Amplitude Estimators ஆகியவை உள்ளன. மேலே உள்ள அல்காரிதம்ஸ் ஆவணமாக்கல் இணைப்பில் வகைகளின் முழு தொகுப்பையும் காணலாம்.

வழிமுறைகள் உள்ளமைக்கக்கூடியவை மற்றும் பெரும்பாலும் உள்ளமைவின் ஒரு பகுதி சிறிய கட்டுமானத் தொகுதிகளின் வடிவத்தில் இருக்கும், அவற்றில் கட்டிடத் தொகுதி வகையின் வெவ்வேறு நிகழ்வுகளை வழங்க முடியும். உதாரணமாக, VQE, மாறுபட்ட குவாண்டம் ஐஜென்சால்வர், இது ஒரு சோதனை அலைவடிவத்தை QuantumCircuit மற்றும் பிறவற்றில் கிளாசிக்கல் ஆப்டிமைசர் வடிவத்தில் எடுக்கிறது.

VQE உதாரணத்தை உருவாக்க ஒரு எடுத்துக்காட்டைப் பார்ப்போம். இங்கே TwoLocal என்பது மாறுபட்ட வடிவம் (சோதனை அலை செயல்பாடு), அளவுருவாக்கப்பட்ட சுற்று, இது மாறுபடும், மற்றும் SLSQP ஒரு கிளாசிக்கல் ஆப்டிமைசர். இவை தனித்தனி நிகழ்வுகளாக உருவாக்கப்பட்டு, அது கட்டப்படும்போது VQE க்கு அனுப்பப்படும். எடுத்துக்காட்டாக, வேறுபட்ட கிளாசிக்கல் ஆப்டிமைசர் அல்லது மாறுபட்ட வடிவத்தை முயற்சிப்பது என்பது நீங்கள் விரும்பும் ஒரு நிகழ்வை உருவாக்கி அதை VQE க்கு அனுப்பும் ஒரு சந்தர்ப்பமாகும்.

[1]:

from qiskit.algorithms import VQE
from qiskit.algorithms.optimizers import SLSQP
from qiskit.circuit.library import TwoLocal

num_qubits = 2
ansatz = TwoLocal(num_qubits, 'ry', 'cz')
opt = SLSQP(maxiter=1000)
vqe = VQE(ansatz, optimizer=opt)

அன்சாட்ஸை வரைவோம், இது ஒரு குவாண்டம் சிர்க்யூட் என்பதைக் காணலாம், அங்கு θ[0] முதல் θ [7] வரை அளவுருக்கள் மாறுபடும், இது VQE ஆப்டிமைசர் குறைந்தபட்ச ஈஜென்வெல்யூவைக் கண்டுபிடிப்பதால் மாறுபடும். கீழேயுள்ள ஒரு வேலை எடுத்துக்காட்டில் நாம் பின்னர் அளவுருக்களுக்கு வருவோம்.

நாம் வழிமுறையை இயக்குவதற்கு முன்பு இன்னும் பல தேவை, எனவே அடுத்ததைப் பார்ப்போம்.

ஒரு வழிமுறையை எவ்வாறு இயக்குவது?¶

ஒரு வழிமுறையை இயக்குவதற்கு, பின்தளத்தில், ஒரு சிமுலேட்டர் அல்லது உண்மையான சாதனம் இருக்க வேண்டும், அதில் வழிமுறையை உள்ளடக்கிய சுற்றுகள் இயக்கப்படலாம். எனவே எடுத்துக்காட்டாக, ஏர் வழங்குநரிடமிருந்து aer_simulator_statevector ஐப் பயன்படுத்தலாம்.

[3]:

from qiskit import Aer

backend = Aer.get_backend('aer_simulator_statevector')

இப்போது ஒரு பின்தளத்தில் நீங்கள் எவ்வாறு சுற்றுகளை இயக்க விரும்புகிறீர்கள் என்பது பற்றிய தகவல்கள் இல்லை. எடுத்துக்காட்டாக, எத்தனை காட்சிகளை, உங்களுக்கு ஒரு சத்தம் மாதிரி வேண்டுமா, சுற்றுகளை மாற்றுவதற்கான விருப்பங்கள் கூட. இதற்காக Qiskit அக்வாவில் QuantumInstance உள்ளது, உதாரணமாக:இது பின்தளத்தில் மற்றும் சுற்றுச் செயலாக்கத்தைச் சுற்றியுள்ள பல்வேறு அமைப்புகளையும் செயல்படுத்துகிறது:

[4]:

from qiskit.utils import QuantumInstance

backend = Aer.get_backend('aer_simulator')
quantum_instance = QuantumInstance(backend=backend, shots=800, seed_simulator=99)

குறிப்பு: நீங்கள் பின்னணியை நேரடியாக தரும் என்றால் நேரடியாக ஒரு குவாண்டம் உதாரணம் உருவாக்கப்படும், முன்னிருப்பு அமைப்புகளுடன், அதனால் அனைத்து நேரத்தில் படிமங்கள் ஒரு குவாண்டம் உதாரணம் வழியாக இயங்குகின்றன.

எனவே இப்போது நாம் compute_mininum_eigenvalue () முறையை அழைக்க முடியும். பிந்தையது நேச்சர் மற்றும் ஆப்டிமைசேஷன் போன்ற பயன்பாட்டு தொகுதிகளுக்கான தேர்வுக்கான இடைமுகமாகும், அவை குறிப்பிட்ட வகைக்குள் எந்த வழிமுறையுடனும் மாறி மாறி செயல்பட முடியும்.

ஒரு முழுமையான பணி உதாரணம்¶

மேலே இருந்து நாம் கற்றுக்கொண்டவற்றை ஒன்றாக இணைத்து ஒரு முழுமையான வேலை உதாரணத்தை உருவாக்குவோம். VQE குறைந்தபட்ச ஈஜென்வெல்யூவைக் கண்டுபிடிக்கும், அதாவது ஒரு ஹாமில்டோனிய ஆபரேட்டரின் குறைந்தபட்ச ஆற்றல் மதிப்பு, எனவே VQE உடன் பணிபுரிய இதுபோன்ற ஒரு ஆபரேட்டர் நமக்குத் தேவை. அத்தகைய ஆபரேட்டர் கீழே கொடுக்கப்பட்டுள்ளது. இது முதலில் வேதியியல் பயன்பாட்டு தொகுதியால் 0.735A இன்டராடோமிக் தூரத்தில் ஒரு H2 மூலக்கூறுக்கு ஹாமில்டோனியனாக உருவாக்கப்பட்டது. இது கீழே உள்ள பவுலி சொற்களின் கூட்டுத்தொகை, ஆனால் இப்போது நான் இதைப் பற்றி மேலும் எதுவும் கூறப்போவதில்லை, ஏனெனில் வழிமுறையை இயக்குவதே குறிக்கோள், ஆனால் ஆபரேட்டர்கள் பற்றிய கூடுதல் தகவல்களை மற்ற பயிற்சிகளில் காணலாம்.

[5]:

from qiskit.opflow import X, Z, I

H2_op = (-1.052373245772859 * I ^ I) + \
        (0.39793742484318045 * I ^ Z) + \
        (-0.39793742484318045 * Z ^ I) + \
        (-0.01128010425623538 * Z ^ Z) + \
        (0.18093119978423156 * X ^ X)

எனவே, குவாண்டம் இன்ஸ்டான்ஸைப் பயன்படுத்தி பின்தளத்தில் கடந்து ஒரு VQE உதாரணத்தை கட்டமைப்பாளருக்கு உருவாக்குவோம். இப்போது VQE க்கு செட்டர்கள் உள்ளன, எனவே குவாண்டம் இன்ஸ்டான்ஸையும் பின்னர் அனுப்பலாம்.

குறிப்பு: நீங்கள் இந்த நோட்புக்கை இயக்கவும், அக்வா_குளோபால்களில் அக்வா முழுவதும் பயன்படுத்தப்படும் சீரற்ற எண் ஜெனரேட்டரைப் பார்க்கவும், அதே போல் குவாண்டம் இன்ஸ்டான்ஸ் வழியாக டிரான்ஸ்பைலர் மற்றும் சிமுலேட்டரை விதைக்கவும் முடியும். நீங்கள் இதைச் செய்ய வேண்டியதில்லை, ஆனால் ஒவ்வொரு முறையும் அதே விளைவை மீண்டும் உருவாக்க முடியும் என்றால், இது இப்படித்தான் செய்யப்படுகிறது.

எனவே VQE ஐ இயக்கி, அது தரும் முடிவு பொருளை அச்சிடுவோம்.

[6]:

from qiskit.utils import algorithm_globals
seed = 50
algorithm_globals.random_seed = seed
qi = QuantumInstance(Aer.get_backend('statevector_simulator'), seed_transpiler=seed, seed_simulator=seed)

ansatz = TwoLocal(rotation_blocks='ry', entanglement_blocks='cz')
slsqp = SLSQP(maxiter=1000)
vqe = VQE(ansatz, optimizer=slsqp, quantum_instance=qi)
result = vqe.compute_minimum_eigenvalue(H2_op)
print(result)

ஆப்டிமைசர் மேலே கொடுக்கப்பட்ட முடிவிலிருந்து, அளவுரு மதிப்புகளின் உகப்பாக்கி 65 மதிப்பீடுகளை எடுத்ததைக் காணலாம், இது -1.85727 இன் குறைந்தபட்ச eigenvalue க் கண்டுபிடிக்கும் வரை, இது கொடுக்கப்பட்ட H2 மூலக்கூறின் மின்னணு நில நிலை ஆற்றலாகும். ansatz ன் உகந்த அளவுருக்கள் ansatz ல் குறைந்தபட்ச மதிப்பில் இருந்த மதிப்புகள் என்பதையும் காணலாம்.

கட்டுமானத்திற்குப் பிறகு குவாண்டம் இன்ஸ்டான்ஸ் அமைத்தல்¶

மூடுவதற்கு, குவாண்டம் இன்ஸ்டான்ஸை வழங்காமல் ஒரு VQE நிகழ்வையும் உருவாக்குவோம். VQE இயங்கும் நேரத்தில் அதைப் பயன்படுத்த ஒரு குவாண்டம் இன்ஸ்டான்ஸ் இருக்கும் வரை நாங்கள் அதை பின்னர் அமைத்துள்ளோம்.
