சத்தத்துடன் ஏர் சிமுலேட்டரில் VQE¶

Qiskit Aer aer_simulator கொடுக்கப்பட்ட இரைச்சல் மாதிரியை அடிப்படையாகக் கொண்டு, சத்தத்துடன் ஒரு உருவகப்படுத்துதலை இயக்க இந்த நோட்புக் நிரூபிக்கிறது. வெவ்வேறு இரைச்சல் நிலைமைகளின் கீழ் நடத்தை விசாரிக்க இது பயனுள்ளதாக இருக்கும். உங்கள் சொந்த தனிப்பயன் இரைச்சல் மாதிரியை வரையறுக்க ஏர் உங்களை அனுமதிப்பது மட்டுமல்லாமல், உண்மையான குவாண்டம் சாதனத்தின் பண்புகளின் அடிப்படையில் இரைச்சல் மாதிரியை எளிதில் உருவாக்க அனுமதிக்கிறது. பிந்தையது இந்த நோட்புக் நிரூபிக்கும் என்பதால், VQE ஐ சத்தத்துடன் காண்பிப்பதே குறிக்கோள், தனிப்பயன் இரைச்சல் மாதிரிகளை எவ்வாறு உருவாக்குவது என்பது மிகவும் சிக்கலான பணி அல்ல.

Qiskit Ignis சத்தமில்லாத உருவகப்படுத்துதல் அல்லது உண்மையான குவாண்டம் சாதனத்தில் இயங்கும் போது அளவீட்டு பிழையைத் தணிக்க ஒரு தீர்வை வழங்குகிறது. இந்த தீர்வு QuantumInstance தொகுதிக்கூறுகளால் பயன்படுத்தப்படுகிறது, இது எந்த வழிமுறையும் பயன்படுத்தும் அளவீட்டு தானாக அளவீட்டு இரைச்சல் குறைப்பை அனுமதிக்கிறது.

Qiskit ஏர் இரைச்சல் மாதிரியைப் பற்றிய கூடுதல் தகவல்களை ஆன்லைனில் Qiskit Aer documentation, இல் காணலாம், மேலும் அதற்கான பயிற்சி building noise models.

Qiskit இக்னிஸில் அளவீட்டு பிழை குறைப்பு பற்றிய கூடுதல் தகவல்களை measurement error mitigation. க்கான பயிற்சியில் காணலாம்.

import numpy as np
import pylab

from qiskit import Aer

from qiskit.circuit.library import TwoLocal
from qiskit.opflow import I, X, Z

ஒரு ஹாமில்டோனியனின் குறைந்தபட்ச (தரை நிலை) ஆற்றலைக் கண்டுபிடிக்கும் VQE உடன் சத்தம் உருவகப்படுத்துதல் இங்கே நிரூபிக்கப்படும், ஆனால் கிஸ்கிட்டிலிருந்து எந்த குவாண்டம் அல்காரிதத்திற்கும் இந்த நுட்பம் பொருந்தும்.

மூலக்கூறுஎனவே VQE க்கு உள்ளீடாக ஒரு குயுபிட் ஆபரேட்டர் தேவை. இங்கே, மீண்டும், H2 மூலக்கூறுக்காக, Qiskit வேதியியலால் முதலில் கணக்கிடப்பட்ட பாலிஸின் தொகுப்பை எடுப்போம், எனவே விரைவாக ஒரு ஆபரேட்டரை உருவாக்க முடியும்.

print(f'Number of qubits: {H2_op.num_qubits}')

*ஒலி இல்லாமல் செயல் *¶

முதலில் முடிவைக் காண சத்தம் சேர்க்காமல் சிமுலேட்டரை இயக்குவோம். நான் பின்தளத்தில் மற்றும் குவாண்டம்இன்ஸ்டான்ஸை உருவாக்கியுள்ளேன், இது பின்தளத்தில் மற்றும் பல ரன் நேர உள்ளமைவுகளை வைத்திருக்கிறது, அவை இங்கு இயல்புநிலையாக உள்ளன, எனவே சத்தம் சேர்க்கப்படும் அடுத்த பகுதிக்கு வரும்போது ஒப்பிடுவது எளிது. இந்த நோட்புக்கில் சத்தம் அல்லது எதையும் குறைக்க எந்த முயற்சியும் இல்லை, எனவே VQE இன் பிந்தைய அமைவு மற்றும் இயக்கம் ஒரே மாதிரியாக இருக்கும்.

counts = []
values = []

    values.append(mean)

result = vqe.compute_minimum_eigenvalue(operator=H2_op)

சத்தத்துடன் செயல்திறன்¶

இப்போது சத்தம் சேர்ப்போம். இங்கே ஒரு உண்மையான சாதனத்திலிருந்து ஏர் க்கான இரைச்சல் மாதிரியை உருவாக்குவோம். நீங்கள் ஏர் மூலம் தனிப்பயன் இரைச்சல் மாதிரிகளை உருவாக்கலாம், ஆனால் அது இந்த நோட்புக்கின் எல்லைக்கு அப்பாற்பட்டது. ஏர் இரைச்சல் மாதிரியைப் பற்றிய கூடுதல் தகவலுக்கான இணைப்புகள், இதைச் செய்ய ஆர்வமுள்ளவர்களுக்கு மேலே கொடுக்கப்பட்டுள்ளன.

முதலில் நாம் ஒரு உண்மையான சாதனை பின்னணி பெற வேண்டும் மற்றும் அதன் configuration மற்றும் properties நாம் ஒரு கூட்டு வரைபடம் மற்றும் ஒரு ஒளி மாதிரி சாதனை பொருந்தும். நாம் அனைத்து வரைபடங்களையும் முன்னிருப்பாக விட்டு விட்டு செல்லும் போது, இந்த காட்டுகிறது கூம்பு வரைபடத்தை எப்படி அமைக்க எப்படி. குறிப்பு: நாம் தேர்வு செய்யப்பட்டால் வேறுபடி வடிவத்துக்காக இந்த கூம்பு வரைபடத்தை பயன்படுத்த முடியும்.

குறிப்பு: சத்தத்துடன் உருவகப்படுத்துதல் சத்தம் இல்லாமல் இருப்பதை விட அதிக நேரம் எடுக்கும்.

சத்தம் மற்றும் அளவீட்டு பிழை குறைப்புடன் செயல்திறன்¶

இப்போது அளவீட்டு பிழை குறைப்புக்கான முறையைச் சேர்ப்போம், இது அளவீட்டின் நம்பகத்தன்மையை அதிகரிக்கிறது. அளவீட்டு பிழையைத் தணிக்க CompleteMeasFitter ஐ இங்கே தேர்வு செய்கிறோம். அளவுத்திருத்த அணி ஒவ்வொரு 30 நிமிடங்களுக்கும் (இயல்புநிலை மதிப்பு) தானாக புதுப்பிக்கப்படும்.

குறிப்பு: சத்தத்துடன் உருவகப்படுத்துதல் சத்தம் இல்லாமல் இருப்பதை விட அதிக நேரம் எடுக்கும்.

முடிவுகளை இங்கே சுருக்கமாகக் கொண்டு வரலாம்.

நாங்கள் ஒரு கிளாசிக்கல் வழிமுறையைப் பயன்படுத்தி ஒரு குறிப்பு மதிப்பை உருவாக்கி, பின்னர் ஒரு Qasm சிமுலேட்டரில் VQE ஐ இயக்கத் தொடங்கினோம். உருவகப்படுத்துதல் சிறந்தது என்றாலும் (சத்தம் இல்லை) மாதிரி காரணமாக ஷாட்-இரைச்சல் என்று அழைக்கப்படுகிறது - காட்சிகளின் எண்ணிக்கையை அதிகரிப்பது மேலும் மேலும் மாதிரிகள் எடுக்கப்படுவதால் இதைக் குறைக்கிறது, ஆனால் காட்சிகள் 1024 இன் இயல்புநிலையில் விடப்பட்டன, மேலும் ஒரு சிறிய விளைவைக் காண்கிறோம் முடிவில்.

உண்மையான சாதனத்திலிருந்து எடுக்கப்பட்ட மாதிரியைப் பயன்படுத்தி சத்தத்தைச் சேர்த்துள்ளோம், இதன் விளைவாக பாதிக்கப்படுவதைக் காணலாம். இறுதியாக அளவீட்டு இரைச்சலைக் குறைப்பதைச் சேர்த்துள்ளோம், இது கிளாசிட்களை அளவிடும் கிளாசிக்கல் கருவிகளில் சத்தத்தின் தாக்கத்தைத் தணிக்கும் முயற்சியில் முடிவுகளை சரிசெய்கிறது.
