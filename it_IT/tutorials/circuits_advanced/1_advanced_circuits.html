Gates Composti¶

# Build a sub-circuit
sub_q = QuantumRegister(2)
sub_circ = QuantumCircuit(sub_q, name='sub_circ')
sub_circ.h(sub_q[0])
sub_circ.crz(1, sub_q[0], sub_q[1])
sub_circ.barrier()
sub_circ.id(sub_q[1])
sub_circ.u3(1, 2, -2, sub_q[0])

# Convert to a gate and stick it into an arbitrary place in the bigger circuit
sub_inst = sub_circ.to_instruction()

qr = QuantumRegister(3, 'q')
circ = QuantumCircuit(qr)
circ.h(qr[0])
circ.cx(qr[0], qr[1])
circ.cx(qr[1], qr[2])
circ.append(sub_inst, [qr[1], qr[2]])

circ.draw()

Circuiti parametrizzati¶

[6]:

from qiskit.circuit import Parameter

theta = Parameter('θ')

n = 5

qc = QuantumCircuit(5, 1)

qc.h(0)
for i in range(n-1):
    qc.cx(i, i+1)

qc.barrier()
qc.rz(theta, range(5))
qc.barrier()

for i in reversed(range(n-1)):
    qc.cx(i, i+1)
qc.h(0)
qc.measure(0, 0)

qc.draw('mpl')

Associare i parametri ai valori¶

Tutti i parametri del circuito devono essere collegati prima di inviare il circuito a un backend. Questo può essere fatto in due modi: - Il metodo bind_parameters accetta un dizionario di mappatura dei Parameters ai valori e restituisce un nuovo circuito in cui ogni parametro viene sostituito dal suo valore corrispondente. È supportato un binding parziale, nel qual caso il circuito restituito sarà parametrizzato da qualsiasi Parameters che non è stato mappato ad un valore.

[8]:

import numpy as np

theta_range = np.linspace(0, 2 * np.pi, 128)

circuits = [qc.bind_parameters({theta: theta_val})
            for theta_val in theta_range]

circuits[-1].draw()

Ridurre i costi di compilazione¶

La compilazione su un circuito parametrizzato prima del binding può, in alcuni casi, ridurre considerevolmente il tempo di compilazione rispetto alla compilazione su un insieme di circuiti collegati.

[11]:

import time
from itertools import combinations
from qiskit.compiler import assemble
from qiskit.test.mock import FakeVigo

start = time.time()
qcs = []

theta_range = np.linspace(0, 2*np.pi, 32)

for n in theta_range:
    qc = QuantumCircuit(5)

    for k in range(8):
        for i,j in combinations(range(5), 2):
            qc.cx(i,j)
        qc.rz(n, range(5))
        for i,j in combinations(range(5), 2):
            qc.cx(i,j)

    qcs.append(qc)

compiled_circuits = transpile(qcs, backend=FakeVigo())
qobj = assemble(compiled_circuits, backend=FakeVigo())

end = time.time()
print('Time compiling over set of bound circuits: ', end-start)

Time compiling over set of bound circuits:  8.771650075912476

Composizione¶

I circuiti parametrizzati possono essere composti come nel caso di QuantumCircuits standard. In generale, quando si compongono due circuiti parametrizzati, il circuito risultante sarà parametrizzato dall’unione dei parametri dei circuiti di ingresso.

Tuttavia, i nomi dei parametri devono essere univoci all’interno di un determinato circuito. Quando si tenta di aggiungere un parametro il cui nome è già presente nel circuito di destinazione: - se l’origine e la destinazione condividono la stessa istanza Parameter si presume che i parametri siano gli stessi e vengono combinati - se l’origine e la destinazione hanno invece istanze Parameter diverse, verrà generato un errore

[13]:

phi = Parameter('phi')

sub_circ1 = QuantumCircuit(2, name='sc_1')
sub_circ1.rz(phi, 0)
sub_circ1.rx(phi, 1)

sub_circ2 = QuantumCircuit(2, name='sc_2')
sub_circ2.rx(phi, 0)
sub_circ2.rz(phi, 1)

qc = QuantumCircuit(4)
qr = qc.qregs[0]

qc.append(sub_circ1.to_instruction(), [qr[0], qr[1]])
qc.append(sub_circ2.to_instruction(), [qr[0], qr[1]])

qc.append(sub_circ2.to_instruction(), [qr[2], qr[3]])

print(qc.draw())

# The following raises an error: "QiskitError: 'Name conflict on adding parameter: phi'"
# phi2 = Parameter('phi')
# qc.u3(0.1, phi2, 0.3, 0)

Per inserire un sotto circuito con una diversa parametrizzazione, il metodo to_instruction accetta un argomento facoltativo (parameter_map) che, quando presente, genererà istruzioni con il parametro sorgente sostituito da un nuovo parametro.

p = Parameter('p')
qc = QuantumCircuit(3, name='oracle')
qc.rz(p, 0)
qc.cx(0, 1)
qc.rz(p, 1)
qc.cx(1, 2)
qc.rz(p, 2)

theta = Parameter('theta')
phi = Parameter('phi')
gamma = Parameter('gamma')

qr = QuantumRegister(9)
larger_qc = QuantumCircuit(qr)
larger_qc.append(qc.to_instruction({p: theta}), qr[0:3])
larger_qc.append(qc.to_instruction({p: phi}), qr[3:6])
larger_qc.append(qc.to_instruction({p: gamma}), qr[6:9])
print(larger_qc.draw())

print(larger_qc.decompose().draw())
