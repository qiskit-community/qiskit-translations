Utilities (qiskit.providers.aer.utils)¶

This module contains utility functions for modifying NoiseModel objects and QuantumCircuits using noise models.

Classes¶

NoiseTransformer()

Transforms one quantum channel to another based on a specified criteria.

Functions¶
