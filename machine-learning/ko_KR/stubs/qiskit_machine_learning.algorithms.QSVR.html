QSVR¶

class QSVR(*args, quantum_kernel=None, **kwargs)[소스]¶

기반 클래스: sklearn.svm._classes.SVR

Quantum Support Vector Regressor.

This class shows how to use a quantum kernel for regression. The class extends sklearn.svm.SVR, and thus inherits its methods like fit and predict used in the example below. Read more in the sklearn user guide.

Example

qsvr = QSVR(quantum_kernel=qkernel)
qsvr.fit(sample_train,label_train)
qsvr.predict(sample_test)
매개변수

  • quantum_kernel (Optional[QuantumKernel]) – QuantumKernel to be used for regression.

  • *args – Variable length argument list to pass to SVR constructor.

  • **kwargs – Arbitrary keyword arguments to pass to SVR constructor.

Attributes
