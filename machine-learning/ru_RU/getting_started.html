Начало работы¶

Инструкция по установке¶

Пакет Qiskit Machine Learning зависит от основного пакета Qiskit, который имеет свою собственную страницу ` Qiskit Getting Started <https://qiskit.org/documentation/getting_started.html>` __ с подробным описанием опций установки Qiskit и поддерживаемых сред/платформ. Обратите на это внимание в первую очередь. Далее здесь можно найти информацию, которая сфокусируется на установке, специфичной для компонента Qiskit Machine Learning.

В Qiskit Machine Learning есть некоторые функции, которые были сделаны необязательными, если зависимые программы и/или программы поддержки не установлены (или не могут быть установлены) по умолчанию. Это PyTorch и Sparse. Дополнительная информация приведена в разделе :ref:` optional_installs `.

Самый простой способ - начать с раздела „Запуск локально“ для Qiskit здесь Qiskit Getting Started

В Вашей виртуальной среде, где Вы установили Qiskit, просто добавьте machine-learning в список дополнительных пакетов подобно тому как устанавливается дополнительная поддержка visualization в Qiskit, например:

pip install qiskit[machine-learning]

Стоит отметить, что если Вы используете zsh (оболочка по умолчанию для более новых версий macOS), то Вам нужно поставить qiskit[machine-learning] в кавычки:

pip install 'qiskit[machine-learning]'

Установка элементов из исходного кода позволит Вам получить доступ к последней обновленной версии Qiskit Machine Learning вместо использования версии из репозитория Python Package Index (PyPI). Это даст Вам возможность более эффективно изучить и внести свой вклад в улучшение последней версии Qiskit Machine Learning кода.

Так как Qiskit Machine Learning зависит от Qiskit, и его последние изменения могут потребовать новых или измененных функций Qiskit, сначала следуйте инструкциям «Установить из исходного кода» здесь Qiskit Getting Started

Installing Qiskit Machine Learning from Source

Используя ту же среду разработки, в которой Вы установили Qiskit, Вы можете установить пакет Qiskit Machine Learning.

  1. Клонируйте репозиторий Qiskit Machine Learning.

    git clone https://github.com/Qiskit/qiskit-machine-learning.git

  2. Клонирование репозитория создает локальную папку с названием qiskit-machine-learning.

    cd qiskit-machine-learning

  3. Если Вы хотите запустить тесты, установите необходимые инструменты для разработчика.

    pip install -r requirements-dev.txt

  4. Установите qiskit-machine-learning.

    pip install .

Если Вы хотите установить пакет в редактируемом режиме, который позволяет не переустанавливать пакет в случае изменения кода в проекте, Вы можете сделать это с помощью:

pip install -e .

Необязательные для установки компоненты¶

  • PyTorch можно установить либо с помощью команды pip install „qiskit-machine-learning[torch]“ для установки пакета, либо обратитесь на страницу [с чего начать](https://pytorch.org/get-started/locally/) сайта PyTorch. Когда PyTorch установлен, TorchConnector облегчает использование квантово вычисляемых сетей.

  • Sparse может быть установлен с помощью команды pip install „qiskit-machine-learning[sparse]“. Sparse позволяет использовать разреженные массивы/тензоры.
