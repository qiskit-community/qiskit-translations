Primeros pasos¶

Instalación¶

Qiskit Machine Learning depende del paquete principal de Qiskit, que tiene su propio Qiskit Primeros Pasos que detalla las opciones de instalación de Qiskit y sus entornos/plataformas compatibles. Deberías referirte a eso primero. Luego, se puede seguir la información que está aquí la cual se centra en la instalación adicional específica de Qiskit Machine Learning.

Qiskit Machine Learning tiene algunas funciones que se han hecho opcionales cuando el código dependiente y/o los programa(s) de soporte no están (o no pueden ser) instalados de forma predeterminada. Esos son PyTorch y Sparse. Consulte Instalaciones opcionales para obtener más información.

La forma más sencilla de empezar es seguir la introducción “Comenzar localmente” para Qiskit que se encuentra aquí Qiskit Primeros Pasos

En tu entorno virtual donde instalaste Qiskit, simplemente agrega machine-learning a la lista adicional de manera similar a cómo se instala el soporte adicional de visualization para Qiskit, es decir:

pip install qiskit[machine-learning]

Vale la pena señalar que si eres un usuario de zsh (que es el shell predeterminado en las versiones más recientes de macOS), deberás poner qiskit[machine-learning] entre comillas:

pip install 'qiskit[machine-learning]'

La instalación de Qiskit Machine Learning desde el código fuente te permite acceder a la versión actualizada más recientemente en desarrollo en lugar de usar la versión en el repositorio Python Package Index (PyPI). Esto te dará la capacidad de inspeccionar y ampliar la última versión del código de Qiskit Machine Learning de manera más eficiente.

Dado que Qiskit Machine Learning depende de Qiskit, y sus últimos cambios pueden requerir funciones nuevas o modificadas de Qiskit, primero debe seguir las instrucciones de «Instalar desde el código fuente» de Qiskit aquí Qiskit Primeros Pasos

Installing Qiskit Machine Learning from Source

Con el mismo entorno de desarrollo en el que instalaste Qiskit, estás listo para instalar Qiskit Machine Learning.

  1. Clona el repositorio de Qiskit Machine Learning.

    git clone https://github.com/Qiskit/qiskit-machine-learning.git

  2. La clonación del repositorio crea una carpeta local llamada qiskit-machine-learning.

    cd qiskit-machine-learning

  3. Si deseas ejecutar pruebas o verificaciones de linting, instala los requerimientos del desarrollador.

    pip install -r requirements-dev.txt

  4. Instala qiskit-machine-learning.

    pip install .

Si deseas instalarlo en modo editable, lo que significa que los cambios de código en el proyecto no requieren una reinstalación para ser aplicados, puedes hacerlo con:

pip install -e .

Instalaciones opcionales¶

  • PyTorch, se puede instalar usando el comando pip install “qiskit-machine-learning[torch]” para instalar el paquete o consultar PyTorch [getting started](https://pytorch.org/get-started/locally/). Cuando se instala PyTorch, el TorchConnector facilita su uso de redes de computación cuántica.

  • Sparse, se puede instalar usando el comando pip install “qiskit-machine-learning[sparse]” para instalar el paquete. La instalación de sparse permitirá el uso de arreglos/tensores dispersos.
