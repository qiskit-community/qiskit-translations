Εγκατάσταση¶

Η Μηχανική Εκμάθηση Qiskit εξαρτάται από το κύριο πακέτο Qiskit που έχει το δικό του «Qiskit Getting Started <https://qiskit.org/documentation/getting_started.html>«__ που περιγράφει λεπτομερώς τις επιλογές εγκατάστασης για το Qiskit και τα υποστηριζόμενα περιβάλλοντα/πλατφόρμες. Στη συνέχεια, οι πληροφορίες εδώ μπορεί να ακολουθηθεί η οποία επικεντρώνεται στην πρόσθετη εγκατάσταση ειδικά για την Qiskit μάθηση μηχανής.

Η Μηχανική Εκμάθηση Qiskit έχει κάποιες λειτουργίες που έχουν γίνει προαιρετικές όπου ο εξαρτώμενος κώδικας ή/και το πρόγραμμα(τα) υποστήριξης δεν είναι (ή δεν μπορεί να είναι) εγκατεστημένο από προεπιλογή. Αυτά είναι τα PyTorch και Sparse. Δείτε το Προαιρετικές εγκαταστάσεις για περισσότερες πληροφορίες.

Ο απλούστερος τρόπος για να ξεκινήσετε είναι να ακολουθήσετε το “Ξεκινήστε τοπικά” για Qiskit εδώ ` Qiskit Ξεκινώντας <https://qiskit.org/documentation/getting_started.html>` __ _

Στο εικονικό περιβάλλον σας, όπου εγκαταστήσατε το Qiskit απλά προσθέστε το machine-learning στην επιπλέον λίστα με παρόμοιο τρόπο όπως εγκαθίσταται η επιπλέον υποστήριξη visualization για το Qiskit, δηλαδή:

pip install qiskit[machine-learning]

Αξίζει να σημειωθεί ότι αν είστε ένας χρήστης zsh (που είναι το προεπιλεγμένο κέλυφος σε νεότερες εκδόσεις του MacOS), θα πρέπει να βάλετε ` ` qiskit[machine-learning]` ` σε εισαγωγικά:

pip install 'qiskit[machine-learning]'

Η εγκατάσταση του Qiskit Machine Learning από την πηγή, σας επιτρέπει να αποκτήσετε πρόσβαση στην πιο πρόσφατα ενημερωμένη υπό ανάπτυξη έκδοση αντί να χρησιμοποιήσετε την έκδοση στο αποθετήριο του Python Package Index (PyPi). Αυτό θα σας δώσει τη δυνατότητα να επιθεωρήσετε και να επεκτείνετε πιο αποτελεσματικά την τελευταία έκδοση του κώδικα μηχανικής μάθησης Qiskit.

Δεδομένου ότι Qiskit Machine Learning εξαρτάται από Qiskit, και οι τελευταίες αλλαγές του μπορεί να απαιτήσουν νέα ή τροποποιημένα χαρακτηριστικά του Qiskit, θα πρέπει πρώτα να ακολουθήσετε τις οδηγίες του Qiskit «Install from source» εδώ Qiskit Getting Started

Installing Qiskit Machine Learning from Source

Χρησιμοποιώντας το ίδιο περιβάλλον ανάπτυξης που εγκαταστήσατε το Qiskit, είστε έτοιμοι να εγκαταστήσετε το Qiskit Machine Learning.

  1. Κλωνοποιήστε το αποθετήριο Qiskit Μηχανικής Εκμάθησης.

    git clone https://github.com/Qiskit/qiskit-machine-learning.git

  2. Η κλωνοποίηση του αποθετηρίου δημιουργεί έναν τοπικό φάκελο που ονομάζεται qiskit-machine-learning.

    cd qiskit-machine-learning

  3. Αν θέλετε να εκτελέσετε δοκιμές ή να επιβάλετε ελέγχους, εγκαταστήστε τις απαιτήσεις ανάπτυξης.

    pip install -r requirements-dev.txt

  4. Εγκατάσταση » ` qiskit-learning-learning ` `.

    pip install .

Αν θέλετε να το εγκαταστήσετε σε επεξεργάσιμη λειτουργία, που σημαίνει ότι οι αλλαγές κώδικα στο έργο δεν απαιτούν την εφαρμογή επανεγκατάστασης, μπορείτε να το κάνετε με:

pip install -e .

Προαιρετικές εγκαταστάσεις¶

  • ** PyTorch**, μπορεί να εγκατασταθεί είτε χρησιμοποιώντας την εντολή ` pip install “qiskit-learning[torch]” “ για να εγκαταστήσετε το πακέτο ή να ανατρέξετε στο PyTorch [ξεκινώντας] (https: //pytorch.org/get-started/locally/). Όταν εγκατασταθεί το PyTorch, το «TorchConnector» διευκολύνει τη χρήση των κβαντικών υπολογιζόμενων δικτύων.

  • Sparse, μπορεί να εγκατασταθεί χρησιμοποιώντας την εντολή pip install “qiskit-machine-learning[sparse]” για να εγκαταστήσετε το πακέτο. Το Sparse που εγκαθίσταται θα ενεργοποιήσει τη χρήση αραιών πινάκων/tensors.
