Kurulum¶

Qiskit Nature; Qiskit için kurulum seçeneklerini ve onun desteklenen ortam/platformlarını detaylandıran Qiskit Başlarken adındaki kendi ana Qiskit paketine bağlıdır. Öncelikle buna bakmalısınız. Sonrasında, burada yer alan Qiskit Nature’a özgü ek kuruluma ilişkin bilgiler takip edilebilir.

Başlamanın en kolay yolu, Qiskit için ‘Yerel Olarak Başlatma’ bölümünü buradan takip ederek başlamaktır: Qiskit Başlarken

Qiskit’i kurduğunuz sanal ortamınızda, Qiskit için ekstra “görselleştirme” desteğinin nasıl kurulduğuna benzer şekilde, “finance”ı ekstra listeye eklemeniz yeterlidir, yani:

pip install qiskit[finance]

Bir zsh kullanıcısıysanız (macOS’un daha yeni sürümlerinde varsayılan kabuktur), ``qiskit[finance]``ı tırnak içine koymanız gerekeceğini belirtmekte fayda var:

pip install 'qiskit[finance]'

Qiskit Finance’ı kaynaktan yüklemek, Python Paket Dizini (PyPI) deposundaki sürümü kullanmak yerine, geliştirilmekte olan en son güncellenen sürüme erişmenizi sağlar. Bu size Qiskit Finance kodunun en son sürümünü daha verimli bir şekilde inceleme ve genişletme kabiliyeti verecektir.

Qiskit Finance, Qiskit’e bağlı olduğundan ve onun son değişiklikleri Qiakit’in yeni ya da değiştirilmiş özelliklerini gerektirebileceğinden, ilk olarak burayı takip etmeniz gerekmektedir: “Install from source” instructions here Qiskit Getting Started

Installing Qiskit Finance from Source

Qiskit’i kurduğunuz aynı geliştirme ortamını kullanarak Qiskit Finance’ı kurmaya hazırsınız.

  1. Qiskit Finance deposunu klonla.

    git clone https://github.com/Qiskit/qiskit-finance.git

  2. Depoyu klonlamak, qiskit-nature adında yerel bir klasör oluşturur.

    cd qiskit-finance

  3. Test veya linting (tiftikleme) kontrolleri yapmak istiyorsanız, geliştirici gereksinimlerini yükleyin.

    pip install -r requirements-dev.txt

  4. qiskit-finance yükleyin.

    pip install .

Düzenlenebilir modda yüklemek istiyorsanız, yani projedeki kod değişikliklerinin yeniden yüklenmesi gerekmiyorsa, bunu şu şekilde yapabilirsiniz:

pip install -e .
