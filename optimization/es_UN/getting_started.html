Primeros Pasos¶

Instalación¶

Qiskit Optimization depende del paquete principal de Qiskit, que tiene su propio Qiskit Primeros Pasos que detalla las opciones de instalación de Qiskit y sus entornos/plataformas compatibles. Deberías referirte a eso primero. Luego, se puede seguir la información que está aquí la cual se centra en la instalación adicional específica de Qiskit Optimization.

Qiskit Optimization tiene algunas funciones que se han hecho opcionales cuando el código dependiente y/o los programa(s) de soporte no están (o no pueden ser) instalados de forma predeterminada. Esos son IBM CPLEX, CVXPY y Matplotlib. Consulte Instalaciones opcionales para obtener más información.

La forma más sencilla de empezar es seguir la introducción “Comenzar localmente” para Qiskit que se encuentra aquí Qiskit Primeros Pasos

En tu entorno virtual donde instalaste Qiskit, simplemente agrega optimization a la lista adicional de manera similar a cómo se instala el soporte adicional de visualization para Qiskit, es decir:

pip install qiskit[optimization]

It is worth pointing out that if you’re a zsh user (which is the default shell on newer versions of macOS), you’ll need to put qiskit[optimization] in quotes:

pip install 'qiskit[optimization]'

La instalación de Qiskit Optimization desde el código fuente te permite acceder a la versión actualizada más recientemente en desarrollo en lugar de usar la versión en el repositorio Python Package Index (PyPI). Esto te dará la capacidad de inspeccionar y ampliar la última versión del código de Optimization de manera más eficiente.

Dado que Qiskit Optimization depende de Qiskit, y sus últimos cambios pueden requerir funciones nuevas o modificadas de Qiskit, primero debe seguir las instrucciones de «Instalar desde el código fuente» de Qiskit aquí Qiskit Primeros Pasos

Installing Qiskit Optimization from Source

Con el mismo entorno de desarrollo en el que instalaste Qiskit, estás listo para instalar Qiskit Optimization.

  1. Clonar el repositorio de Qiskit Optimization.

    git clone https://github.com/Qiskit/qiskit-optimization.git

  2. Cloning the repository creates a local folder called qiskit-optimization.

    cd qiskit-optimization

  3. Si deseas ejecutar pruebas o verificaciones de linting, instala los requerimientos del desarrollador.

    pip install -r requirements-dev.txt

  4. Instalar qiskit-optimization.

    pip install .

Si deseas instalarlo en modo editable, lo que significa que los cambios de código en el proyecto no requieren una reinstalación para ser aplicados, puedes hacerlo con:

pip install -e .

Instalaciones opcionales¶

  • ** IBM CPLEX* * puede ser instalado utilizando pip install “qiskit-optimization[cplex]” para permitir la lectura de archivos LP y el uso de CplexOptimizer para cplex.Cplex.

  • **CVXPY* *, se puede instalar utilizando el comando ` pip install “qiskit-optimization[cvx]” ` para instalar el paquete. Que CVXPY esté instalado habilitará el uso del algoritmo Goemans-Williamson como optimizador ` GoemansWilliamsonOptimizer `.

  • ** Matplotlib**, puede ser instalado utilizando el comando pip install “qiskit-optimization[matplotlib]” para instalar el paquete. La instalación de Matplotlib permitirá el uso del método draw en las clases de aplicación de optimización de gráficos.

  • Gurobipy may be installed using command pip install “qiskit-optimization[gurobi]” to install the package. Gurobipy being installed will enable the usage of the GurobiOptimizer.
