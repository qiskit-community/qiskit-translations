AerJob¶

class AerJob(backend, job_id, fn, qobj, *args)[sumber]¶

AerJob class.

_executor¶

executor to handle asynchronous jobs

Type

futures.Executor

Initializes the asynchronous job.

Parameter

  • backend -- the backend used to run the job.

  • job_id -- a unique id in the context of the backend used to run the job.

Methods

AerJob.backend()

Return the instance of the backend used for this job.

AerJob.cancel()

Attempt to cancel the job.

AerJob.cancelled()

Return whether the job has been cancelled.

AerJob.done()

Return whether the job has successfully run.

AerJob.in_final_state()

Return whether the job is in a final job state.

AerJob.job_id()

Return a unique id identifying the job.

AerJob.qobj()

Return the Qobj submitted for this job.

AerJob.result([timeout])

Get job result.

AerJob.running()

Return whether the job is actively running.

AerJob.status()

Gets the status of the job by querying the Python's future

AerJob.submit()

Submit the job to the backend for execution.

AerJob.wait_for_final_state([timeout, wait, ...])

Poll the job status until it progresses to a final state such as DONE or ERROR.
