Mulai dengan Qiskit¶

Alur kerja dalam menggunakan Qiskit terdiri dari tiga langkah umum:

  • Build: mendesain sebuah sirkuit kuantum yang merepresentasikan masalah yang sedang Anda tinjau.

  • ** Execute**: menjalankan eksperimen pada backends yang berbeda (* yang mencakup sistem dan simulator *).

  • Analyze: menghitung ringkasan statistik dan memvisualisasikan hasil eksperimen.

Berikut ini adalah contoh dari keseluruhan alur kerja, dengan setiap langkah akan dijelaskan secara rinci pada bagian berikutnya:

import numpy as np
from qiskit import(
  QuantumCircuit,
  execute,
  Aer)
from qiskit.visualization import plot_histogram

# Use Aer's qasm_simulator
simulator = Aer.get_backend('qasm_simulator')

# Create a Quantum Circuit acting on the q register
circuit = QuantumCircuit(2, 2)

# Add a H gate on qubit 0
circuit.h(0)

# Add a CX (CNOT) gate on control qubit 0 and target qubit 1
circuit.cx(0, 1)

# Map the quantum measurement to the classical bits
circuit.measure([0,1], [0,1])

# Execute the circuit on the qasm simulator
job = execute(circuit, simulator, shots=1000)

# Grab results from the job
result = job.result()

# Returns counts
counts = result.get_counts(circuit)
print("\nTotal count for 00 and 11 are:",counts)

# Draw the circuit
circuit.draw()

# Plot a histogram
plot_histogram(counts)

Langkah Alur Kerja¶

Program di atas dapat dibagi menjadi enam langkah:

  1. Impor paket

  2. Menginisialisasi variabel

  3. Menambah gate

  4. Visualisasi rangkaian sirkuit

  5. Mensimulasikan eksperimen

  6. Memvisualisasikan hasil

Langkah 1: Impor Paket¶

Elemen dasar yang dibutuhkan untuk program Anda dapat diimpor sebagai berikut:

import numpy as np
from qiskit import(
  QuantumCircuit,
  execute,
  Aer)
from qiskit.visualization import plot_histogram

Secara lebih rinci, yang diimpor adalah

  • ` ` QuantumCircuit ` `: dapat dianggap sebagai instruksi dari sistem kuantum. Ini yang mengatur semua operasi kuantum Anda.

  • ` ` execute ` `: menjalankan rangkaian sirkuit/eksperimen Anda.

  • Aer: menangani backend simulator.

  • plot_histogram: membuat histogram.

Langkah 2: Menginisialisasi Variabel¶

Perhatikan baris kode berikut

circuit = QuantumCircuit(2, 2)

Di sini, anda melakukan inisialisasi dengan 2 qubit dari keadaan nol; dengan 2 bit yang juga dari keadaan nol dan circuit adalah sirkuit kuantum.

Sintaksis:

  • QuantumCircuit(int, int)

Langkah 3: Menambahkan Gates (Kuantum)¶

Anda dapat menambahkan gate (operasi) untuk memanipulasi register di sirkuit Anda.

Perhatikan tiga baris kode berikut:

circuit.h(0)
circuit.cx(0, 1)
circuit.measure([0,1], [0,1])

Gate ditambahkan ke sirkuit satu-per-satu untuk membentuk Bell state

\[|\psi\rangle = \left(|00\rangle+|11\rangle\right)/\sqrt{2}.\]

Kode di atas menerapkan gate berikut:

  • QuantumCircuit.h(0): Sebuah gate Hadamard \(H\) pada qubit 0, menempatkan qubit ini ke dalam keadaan superposisi.

  • QuantumCircuit.cx(0, 1): Sebuah operasi Kontrol Not (\(CNOT\)) pada kontrol di qubit ke 0 dan target di qubit ke 1, menempatkan kedua qubit dalam keadaan entangled.

  • QuantumCircuit.measure([0, 1], [0, 1]): jika Anda mengalirkan seluruh register kuantum dan klasik untuk mengukur, hasil pengukuran qubit ke i akan disimpan dalam bit ke i.

Langkah 4: Visualisasi Sirkuit¶

Anda dapat menggunakan :meth:` qiskit.circuit.QuantumCircuit.draw ` untuk melihat sirkuit yang telah Anda rancang di berbagai bentuk yang digunakan di buku teks dan artikel penelitian.

circuit.draw()

q_0: ┤ H ├──■──┤M├───
     └───┘┌─┴─┐└╥┘┌─┐
q_1: ─────┤ X ├─╫─┤M├
          └───┘ ║ └╥┘
c: 2/═══════════╩══╩═
                0  1

Dalam sirkuit ini, qubit-qubit disusun dengan qubit ke nol di atas dan qubit ke satu di bawahnya. Sirkuit ini dibaca dari kiri ke kanan, yang berarti gate yang diterapkan lebih dahulu muncul lebih ke kiri.

Metoda standar untuk QuantumCircuit.draw() atau qiskit.visualization.circuit_drawer() adalah Ascii teks. Namun, tergantung pada environment lokal yang digunakan, Anda mungkin ingin mengubah ini menjadi sesuatu yang lebih cocok. Ini dapat diset di file konfigurasi pengguna. File konfigurasi pengguna ada di ~/.qiskit/settings.conf dan merupakan file .ini.

Sebagai contoh, settings.conf untuk pengaturan Matplotlib drawer adalah:

[default]
circuit_drawer = mpl

Anda dapat menggunakan salah satu dari metoda visualisasi sirkuit yang valid sebagai nilai di konfigurasi ini, ini termasuk text, mpl, latex, dan latex_source.

Langkah 5: Simulasi Eksperimen¶

Qiskit Aer adalah sebuah framework simulator performa tinggi untuk sirkuit kuantum. Qiskit Aer menyediakan beberapa backend untuk tujuan simulasi yang berbeda.

Jika Anda punya masalah menginstal Aer, Anda dapat menggunakan Basic Aer sebagai alternatif dengan cara mengganti Aer dengan BasicAer. Basic Aer sudah termasuk dalam Qiskit Terra.

import numpy as np
from qiskit import(
  QuantumCircuit,
  execute,
  BasicAer)
...

Untuk mensimulasi sirkuit ini, Anda akan menggunakan qasm_simulator. Setiap kali dieksekusi, sirkuit ini akan menghasilkan bit 00 atau 11.

simulator = Aer.get_backend('qasm_simulator')
job = execute(circuit, simulator, shots=1000)
result = job.result()
counts = result.get_counts(circuit)
print("\nTotal count for 00 and 11 are:",counts)


Total count for 00 and 11 are: {'00': 487, '11': 513}

Seperti yang diharapkan, output bit adalah 00 kira-kira 50 persen dari total. Jumlah eksekusi sirkuit dapat diatur melalui argumen shots dari metode execute. Jumlah tembakan/shots dari simulasi ini diatur menjadi 1000 (pengaturan awalnya adalah 1024).

Setelah Anda memiliki objek result, Anda dapat mengakses counts dengan metode get_counts(circuit). Ini akan memberikan Anda hasil agregat dari percobaan yang Anda jalankan.

Langkah 6: Visualisasi Hasil¶

Qiskit menyediakan berbagai macam visualisasi,

termasuk fungsi plot_histogram, untuk melihat hasil Anda.

plot_histogram(counts)

Probabilitasi \(Pr(00)\) dan \(Pr(11)\) dihitung dengan mengambil counts masing-masing dan dibagi dengan total jumlah tembakan/shots.

Catatan

Cobalah ubah argumen shots pada metode execute untuk melihat bagaimana estimasi probabilitas berubah.
